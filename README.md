# confirm-modal.js
Аппын захиалгыг баталгаажуулах модал.

## open(order, summary)
Захиалгыг баталгаажуулах модалыг нээнэ.
- order: Захиалгын объект.
- summary: Захиалгын барааны дэлгэнгүй объект.
## close()
Захиалгыг баталгаажуулах модалыг хаана.

## confirm event
Хэрэглэгч захиалгыг баталгаажуулахад үүснэ.
## cancel event
Хэрэглэгч захиалгыг цуцлахад үүснэ.

# courier-card.js
Хүргэлтийн ажилтны мэдээллийг харуулах хэсэг.

## setData({ name, phone, student_id, id })
Хүргэлтийн ажилтны мэдээллийг тохируулна.
- name: Хүргэлтийн ажилтны нэр.
- phone: Хүргэлтийн ажилтны утасны дугаар.
- student_id: Хүргэлтийн ажилтны оюутны дугаар.
- id: Хүргэлтийн ажилтны системийн дугаар.

# date-time-picker.js
Огноо, цаг сонгох хэсэг.
## updateToNow()
Одоогийн огноо, цаг руу шинэчилнэ.
## startAutoUpdate()
Одоогийн огноо, цагийг 30 секунд тутамд автоматаар шинэчилнэ.
## get value()
Одоогийн сонгогдсон огноо, цагийг offer-card дээр харагдах шиг хэлбэрээр буцаана.
## get iso()
Одоогийн сонгогдсон огноо, цагийг ISO 8601 хэлбэрээр буцаана.
